# onomatopoeats

Generate memorable random names by combining Japanese onomatopoeia with food words — like `fuwafuwa-mochi` or `sakusaku-tofu`.

## Features

- 70 Japanese onomatopoeia + 60 food nouns = 4,200 combinations
- Separator options: hyphen, underscore, dot
- File extension options: .md, .txt, .json, or none
- One-click copy to clipboard
- History of last 9 generated names
- Pastel color palette with a gentle, stationery-like design
- Pure HTML + CSS + JS — no build step, no dependencies

## Usage

Open `index.html` in a browser, or visit the [GitHub Pages site](https://sumisoc0db8c.github.io/onomatopoeats/).

## Project Structure

```
index.html          HTML shell
style.css           Pastel-themed styles
app.js              Application logic
words/
  onomatopoeia.js   70 onomatopoeia words
  nouns.js          60 Japanese food nouns
```

# onomatopoeats（日本語）

日本語オノマトペと食材名を組み合わせて、覚えやすいランダム名を生成します。
例: `fuwafuwa-mochi`、`sakusaku-tofu`

## 特徴

- オノマトペ 70語 + 食材名 60語 = 4,200通りの組み合わせ
- 区切り文字: ハイフン、アンダースコア、ドット
- 拡張子: .md, .txt, .json, なし
- ワンクリックでクリップボードにコピー
- 直近9件の生成履歴
- やさしいパステルカラーのデザイン
- HTML + CSS + JS のみ — ビルド不要、依存なし

## 使い方

`index.html` をブラウザで開くか、[GitHub Pages](https://sumisoc0db8c.github.io/onomatopoeats/) にアクセスしてください。
